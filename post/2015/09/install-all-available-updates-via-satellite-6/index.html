Install all available updates via Satellite 6

Posted by Christian Jung on Fri, Sep 11, 2015
In CloudForms, IT, ManageIQ, The Foreman
Tags

Red Hat Satellite 6 is a great tool to manage your RHEL virtual machines. In some scenarios you would like to be able to install all available updates for a given system from CloudForms or ManageIQ. I created a small script to accomplish this task.

To use this script, it is necessary to upgrade your Satellite Server to version 6.1.1 or later. Earlier versions do not expose the necessary API calls.

The script which is used here can be found on Github.
